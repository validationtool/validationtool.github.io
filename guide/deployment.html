# Deployment

# aeris-datavalidation-components (VM)

Construction du build

npm run build

Archivage du dist

 tar -cvf dist.tar dist/

Transfert de l'archive vers le serveur

scp dist.tar wwwadm@aeris-recette.ipsl.fr:/tmp

Connexion au serveur (VM)

ssh wwwadm@aeris-recette.ipsl.fr
tar -xvf dist.tar
cd /www/validation/
rm -r *
cp -a /tmp/dist/* .
chmod a+r css/*
chmod a+r  js/*

# aeris-datavalidation-rest

Copier le nouveau jar sur le serveur

scp target/data-validation-rest-X.Y.Z-SNAPSHOT.jar wwwadm@aeris-recette.ipsl.fr:/tmp

Puis aller sur le serveur :

ssh wwwadm@aeris-recette.ipsl.fr
cd /www/actris-validation/services

service actris-validation-rest stop
mv /tmp/data-validation-rest-X.Y.Z-SNAPSHOT.jar actris-validation-rest.jar
chmod 500 actris-validation-rest.jar
service actris-validation-rest start

Ou utiliser le script redeploy.sh (modifier si besoin le numéro de version au début du script, variable VERSION)

Pour vérifier :

service actris-validation-rest status
